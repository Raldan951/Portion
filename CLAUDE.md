## Key Decisions (don't revisit without good reason)
- **Riverpod 3.x**: Use `NotifierProvider`, NOT `StateProvider` (legacy, not exported)
- **KJV default**: First translation shown; principled choice, not just practical
- **Translation tiers**:
  - Tier 1 (bundled): KJV, BSB, WEB ‚Äî local SQLite, no internet needed
  - Tier 2 (API): ESV via api.esv.org ‚Äî internet required
  - Tier 3 (external): Logos deep links ‚Äî `logosres:nkjv;ref=BibleNKJV.Book.Chapter`
- **Logos URL format**: `logosres:{version};ref=Bible{VERSION}.{Book}.{Chapter}` (spaces ‚Üí %20)
- **Morning/Evening labels**: M'Cheyne's "Family/Secret" renamed for clarity
- **Reading plans**: Stored as JSON assets, generated by Python backend
